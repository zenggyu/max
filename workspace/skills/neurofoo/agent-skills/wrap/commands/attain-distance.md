# Attain Distance Before Deciding

Get emotional distance from a decision. Counter short-term emotions that distort judgment.

## Instructions

Use techniques to step back from immediate emotional reactions and connect with your core values and long-term perspective.

### Output Format

**Decision**: [What you're deciding]
**Current Emotional State**: [How you're feeling right now]

## Distance Techniques

**Physical distance**: Have you slept on it?
- [ ] Yes, and my view [changed/stayed the same]
- [ ] No, should I wait?

**Temporal distance**: If this decision were made for you and you found out tomorrow, how would you feel?
[Reaction]

**Social distance**: What would the wisest person you know say?
> [Their likely advice]

## Check: Is More Distance Needed?

- [ ] I can state the decision without strong emotional charge
- [ ] I can see the other side's perspective
- [ ] I'm not deciding out of anger, fear, or desperation
- [ ] I've considered how I'll feel long-term

If unchecked items remain, consider waiting before deciding.
