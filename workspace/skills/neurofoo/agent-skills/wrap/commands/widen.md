## Reframed Decision

**Old frame**: "Should we do X or Y?"

**New frame**: "Given we want [outcome], which of these [N] paths best achieves it?"

## Guidelines

- "Whether or not" is almost always too narrow
- Great decisions come from great options
- You can't choose a better option if you haven't imagined it
- Look for the option you're afraid to consider
