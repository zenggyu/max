# SCAMPER Examples

## Example 1: Reinventing the Coffee Shop

**Subject**: A traditional coffee shop looking to differentiate

---

### S: SUBSTITUTE
| Current | Substitute | Possibility |
|---------|------------|-------------|
| Baristas | Self-serve machines | Lower cost, faster service |
| Coffee | Tea/matcha/adaptogen drinks | Health-conscious positioning |
| Tables and chairs | Standing bars, no seating | Smaller footprint, faster turnover |
| Cash register | App-only ordering | Skip the line |

### C: COMBINE
| Combine A | With B | Result |
|-----------|--------|--------|
| Coffee shop | Coworking space | Pay for time, unlimited coffee |
| Coffee shop | Library | Quiet focus space with drinks |
| Coffee shop | Laundromat | Productive wait time |
| Morning coffee | Fitness | Pre-workout café at gym |

### A: ADAPT
| From | Idea | Application |
|------|------|-------------|
| Chipotle | Assembly line customization | Build-your-own drink bar |
| Wine bars | Tasting flights | Coffee flight samplers |
| Subscription boxes | Recurring delivery | Coffee subscription with café credits |
| Airline lounges | Premium, calm space | Business-focused "executive café" |

### M: MODIFY
| Element | Modification | Effect |
|---------|--------------|--------|
| Cup size | Only one (large) size | Simplicity, premium feel |
| Hours | 24/7 operation | Night owl market |
| Volume | Whisper-quiet policy | Focus-seeking customers |
| Menu | Only 3 drinks | Mastery, faster service |

### P: PUT TO OTHER USES
| Current Use | New Use | Market |
|-------------|---------|--------|
| Morning coffee | Evening dessert destination | Post-dinner crowd |
| Individual drinks | Coffee catering | Office deliveries |
| Café space | Event venue | Private parties, book clubs |
| Barista knowledge | Coffee education | Workshops, certifications |

### E: ELIMINATE
| Eliminate | Result |
|-----------|--------|
| Wifi | Conversation-focused café |
| All seating | Standing-only express |
| Food | Pure coffee focus, lower complexity |
| Tips | Service charge included |
| Menu boards | Surprise drink (trust the barista) |

### R: REVERSE
| Current | Reversed | Possibility |
|---------|----------|-------------|
| Come to café | Café comes to you | Mobile coffee cart |
| Buy what you want | Surprise subscription | Discover new drinks |
| Individual drinks | Family-size carafe | Group-friendly |
| Morning focus | Night-only café | After-dark atmosphere |

**Top Ideas**:
1. Pay-for-time coworking café (combine)
2. Whisper-quiet focus café (modify)
3. Coffee tasting flights (adapt)

## Example 2: Improving Product Onboarding

**Subject**: SaaS product onboarding flow that has high drop-off

---

### S: SUBSTITUTE
| Current | Substitute | Possibility |
|---------|------------|-------------|
| Tutorial videos | Interactive walkthrough | Learn by doing |
| Email drip campaign | In-app progress tracking | More contextual |
| "Getting started" guide | "Quick win" template | Faster value |
| Support chat | AI assistant | 24/7 help |

### C: COMBINE
| Combine A | With B | Result |
|-----------|--------|--------|
| Onboarding | Gamification | Progress badges, streaks |
| Setup | Team invitation | Collaborative start |
| Tutorial | Real work | Use their actual data |
| Learning | Community | Cohort-based onboarding |

### A: ADAPT
| From | Idea | Application |
|------|------|-------------|
| Video games | Tutorial level with rewards | First "mission" to complete |
| Duolingo | Daily streak motivation | Engagement hooks |
| Apple unboxing | Delight in first experience | Premium onboarding design |
| Hotel concierge | Personal success manager call | High-touch segment |

### M: MODIFY
| Element | Modification | Effect |
|---------|--------------|--------|
| Onboarding length | 60 seconds to first value | Reduce time to "aha" |
| Number of steps | From 10 to 3 | Lower cognitive load |
| Personalization | Role-based paths | Relevant first experience |
| Skip option | Make skippable but trackable | Reduce frustration |

### E: ELIMINATE
| Eliminate | Result |
|-----------|--------|
| Account creation | Try before signup |
| Required fields | Minimum viable profile |
| Email verification | Remove friction |
| Tutorial | Learn by exploring (with safety net) |

### R: REVERSE
| Current | Reversed | Possibility |
|---------|----------|-------------|
| Setup then use | Use then setup | Explore first, configure later |
| We teach user | User teaches us | Ask what they want, customize |
| Start empty | Pre-populated template | Immediate value |
| Linear flow | Choose your path | Self-directed discovery |

**Top Ideas**:
1. Pre-populated template for instant value (reverse)
2. Role-based 3-step onboarding (modify)
3. "Quick win" mission with badge (combine + adapt)

## Example 3: Innovation in Insurance

**Subject**: Traditional home insurance product

---

### S: SUBSTITUTE
| Current | Substitute | Possibility |
|---------|------------|-------------|
| Annual premium | Pay-per-use | Seasonal coverage |
| Claims adjusters | Photo-based AI assessment | Faster claims |
| Paper policy | Interactive app dashboard | Real-time coverage view |
| Generic coverage | Personalized by IoT data | Dynamic pricing |

### C: COMBINE
| Combine A | With B | Result |
|-----------|--------|--------|
| Insurance | Smart home devices | Prevention + coverage |
| Homeowners | Neighborhood group | Community policy discounts |
| Insurance | Home warranty | One policy for everything |
| Policy | Savings account | Return unused premium |

### A: ADAPT
| From | Idea | Application |
|------|------|-------------|
| Progressive Snapshot | Home behavior monitoring | Risk-based pricing |
| Lemonade | AI claims bot | Instant claim approval |
| Kickstarter | Peer-to-peer risk pools | Community insurance |
| Netflix | Subscription simplicity | "One price covers all" |

### P: PUT TO OTHER USES
| Current Use | New Use | Market |
|-------------|---------|--------|
| Damage coverage | Prevention services | Loss prevention platform |
| Risk data | Home buying advice | Real estate integration |
| Claim history | Maintenance recommendations | Home care service |

### E: ELIMINATE
| Eliminate | Result |
|-----------|--------|
| Deductible | First-dollar coverage (premium) |
| Paper forms | Digital-only process |
| Physical inspection | Trust + verify with IoT |
| Annual renewal hassle | Auto-renew with review |

### R: REVERSE
| Current | Reversed | Possibility |
|---------|----------|-------------|
| Pay for coverage | Get paid for no claims | Reward prevention |
| Coverage after damage | Prevention before damage | Risk reduction service |
| Annual lump sum | Pay as you risk | Daily/weekly rates |

**Top Ideas**:
1. Smart home + insurance bundle for prevention (combine)
2. Return unused premium to savings (combine)
3. Pay-as-you-risk dynamic pricing (reverse)

## SCAMPER Quick Reference

| Letter | Quick Prompt |
|--------|--------------|
| **S**ubstitute | What else? Who else? What other place/time/material? |
| **C**ombine | What can we merge? What if we bundled? |
| **A**dapt | What can we copy? What's similar elsewhere? |
| **M**odify | Bigger? Smaller? Faster? Slower? Different? |
| **P**ut to other uses | Who else needs this? What other problem? |
| **E**liminate | What's unnecessary? What if we simplified? |
| **R**everse | What if backwards? Opposite? Roles swapped? |
