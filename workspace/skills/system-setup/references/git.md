## Official Documentation

## User Preferences
- Installation method preference: Use apt (system package manager)
- Version preference: Latest stable from official repos

## Definition of Functional
Running `git --version` returns exit code 0 and outputs version string.

## Post-Installation Configuration
- Set global user name and email
- Configure default branch name to `main`
- Set preferred editor
- Enable colored output
- Configure useful aliases (st, co, br, etc.)
- Set up SSH key authentication if not already configured
